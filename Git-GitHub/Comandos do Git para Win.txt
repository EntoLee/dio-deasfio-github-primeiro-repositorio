git config
Defini seu nome de usuário e endereço de e-mail.
git config --global user.name "seu nome"
git config --global user.email seuemail@gmail.com



git init
Cria um repositório em um diretório
git init

Monitora os arquivos
touch .gitignore
git add .gitignore
git commit -m "descrição"

git clone
clona um repositório
git clone git://github.com/user/projeto.git

Clona um repositório em um diretório diferente
git clone git://github.com/user/projeto.git mygrit
